Image Slider Project
A responsive and dynamic image slider built using HTML, CSS, and JavaScript. The project features automatic sliding functionality, manual navigation buttons, and integration with an image array for easy content updates.
Features
Automatic Play: Slides transition automatically every 5 seconds.
Manual Navigation: "Previous" and "Next" buttons for user control.
Array Integration: Images are managed via a JavaScript array for easy addition/removal of slides.
Radio Button Sync: Navigation dots sync with the current slide.

Technologies Used
HTML5: Structure and content.
CSS3: Styling, layout, and transitions.
JavaScript (ES6): Functionality, array management, and event handling.

Getting Started
Follow these instructions to get a copy of the project up and running on your local machine.
Prerequisites
You only need a modern web browser (like Chrome, Firefox, or Edge) to view the project.

Installation.
1.Clone the repository:
bash
git clone github.com
2.Navigate to the project directory:
bash
cd Image-Slider.

3.Open the file:
Open the index.html file in your preferred web browser.
Usage
Adding New Images
To add more images to the slider:
Place your image files (e.g., my_new_pic.jpg) inside your project folder.
Open the script.js file.
Add the filename to the images array.
javascript
const images = [
    'image1.jpg',
    'image2.jpg',
    'image3.jpg',
    'image4.jpg',
    'image5.jpg',
    'my_new_pic.jpg' // Add new image here
];
